SHOPIFY Customer & Sales Analytics

An ETL-ready analytics repository and interactive Power BI / Python starter kit for Shopify order and customer analysis — reusable schemas, example transforms, and dashboard-ready outputs.

Quick start

  1. Clone the repo: git clone https://github.com/Faizan-26079/shopify-sales-analytics.git
  2. Place raw Shopify exports into /data/raw.
  3. Create venv and install deps: pip install -r requirements.txt.
  4. Normalize & export clean files: python scripts/etl_normalize_shopify.py --input data/raw --output data/clean
  5. Open Power BI and point to the CSVs in /data/clean.

Tech stack

Python (pandas, numpy), Power BI, CSV/Parquet, Jupyter notebooks, GitHub Pages.

Repository layout

  • /data — raw and cleaned sample exports (CSV, Parquet).
  • /notebooks — Jupyter notebooks for EDA and cohort analysis.
  • /sql — Reusable SQL analytics queries.
  • /powerbi — .pbix Power BI report template.
  • /scripts — Python ETL helpers and validation scripts.
  • /docs — README images, gif walkthroughs, and schema docs.
